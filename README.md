# Flex でレイアウト構築 実践リポジトリ

VSCode と Live Server を使って、Flexbox のプロパティを実際に書いて体験しよう。

---

## 始め方

1. VSCode でこのフォルダを開く
2. Live Server を起動する（右下の「Go Live」をクリック）
3. `exercise1.html` から始める

---

## 実践①｜親要素プロパティを書いてみよう

**ファイル：** `exercise1.html`

各課題の `<!-- TODO -->` コメントを探して、指示通りのクラスを `class=""` の中に追加する。
保存するたびにブラウザに反映される。

---

## 実践②｜子要素プロパティを自由に試してみよう

**ファイル：** `exercise2.html`

子要素のプロパティを自由に追加・変更して、変化を確認する。
正解はない。気になるものから試してみよう。
